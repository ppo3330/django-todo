# Junseok To-Do App
심플하고 빠르게 할 일을 관리할 수 있는 Django 기반 To-Do 웹 애플리케이션입니다.  
할 일 추가, 수정, 삭제, 체크 기능뿐 아니라 마감일(D-Day)까지 확인할 수 있습니다.

---

# 주요 기능

기본 기능
- 할 일 추가
- 체크박스 통해 완료/미완료 전환
- 수정 / 삭제 기능
- 마감일 설정

D-Day 기능
- D-Day 계산 자동 표시
- 마감일이 지났을 경우 빨간색 경고 표시
- 오늘이 마감일이면 노란색으로 "D-DAY"

필터 기능
- 전체 보기
- 미완료 보기
- 완료된 항목 보기
